# 沙盘三国策略 Demo

原创低多边形 3D 三国风单机策略 Demo。玩法结构参考经典回合策略的“经营城池、出征扩张、统一全图”循环，但地图、势力、武将、文本、数值和视觉素材均为原创，且不依赖外部 3D 模型文件。

## 运行

```bash
npm install
npm run dev
```

打开 `http://127.0.0.1:5188/`。生产构建：

```bash
npm run build
npm run preview
```

## 操作

- 左键选择城池。
- 鼠标拖拽旋转沙盘，滚轮缩放。
- `WASD` 或方向键平移视角。
- 右侧城池面板执行内政、征兵、训练、出征。
- `Space` 结束当前回合。
- `Tab` 切换势力总览。
- `Esc` 关闭弹窗。
- 顶部按钮支持 localStorage 存档与读档。

## 首版内容

- 25 座原创城池，覆盖中原、江东、巴蜀、荆襄、河北、西凉、辽东、南蛮等风格区块。
- 6 个原创势力，玩家初始控制玄麒盟 5 座城。
- 60 名原创武将，含武力、智力、政治、统率、忠诚、体力、技能标签和简介。
- 3D 沙盘地图：地形起伏、山脉、河流、道路、森林、城池、旗帜、选中高亮。
- 回合制经营：农业、商业、城防、民忠、技术、征兵、训练。
- 出征闭环：相邻敌城选择、武将与兵力选择、行军动画、自动战斗结算、归属变化。
- 单挑系统：战前概率触发，使用 3D 小竞技场和 5 回合数值演算表现。
- AI 回合：按粮草、金钱、边境压力和弱敌机会做有限行动，并在控制台输出决策。
- 势力总览、小地图、战报列表、经典/沙盘模式占位、localStorage 存读档。
